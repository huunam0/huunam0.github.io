Tham số hình thức

1. Khái niệm:

+ Hàm (function): là đoạn chương trình (chương trình con) thực hiện một số thao tác và có thể trả về giá trị.

Ví dụ: hàm trả về giá trị lớn nhất của 2 số max(), hàm tính căn bậc hai sqrt(), hàm tính giá trị tuyệt đối abs(),...

Hàm nên độc lập, không phụ thuộc vào bất cứ thứ gì ở bên ngoài.

+ Các biến dùng để mang dữ liệu vào hoặc ra khỏi hàm được gọi là tham số hình thức (hay ngắn gọn là tham số) (parameter)

Ví dụ: hàm tìm số lớn nhất của một mảng

int gtln(int A[], int n)

Tham số hình thức là A, n.

2. Ví dụ 1: hàm tính tổng của 2 số nguyên

#include <iostream>
using namespace std;

int tong(int a, int b)
{
    return a+b;
}

int main()
{
    int x = 5;
    cout << tong(x,7) << endl;
    return 0;
}

3. Ví dụ 2: hàm tăng biến x thêm 1 đơn vị

#include <iostream>
using namespace std;

void tang(int &x)
{
    x = x+1;
}
int main()
{
    int x = 10;
    tang(x);
    cout<<"Gia tri cua x sau khi tang la: "<<x;
    return 0;
}

Lưu ý: khi gọi tang(x) thì TBD sao chép biến x thành biến trùng tên và trùng giá trị, nhưng ở vùng nhớ khác.

Để yêu cầu TBD sử dụng trực tiếp luôn biến x mà không sao chép thì ta thêm kí hiệu & vào trước biến.

Việc dùng trực tiếp biến tham số có 2 lợi ích:

  • Đỡ tốn thời gian sao chép
  • Đỡ tốn bộ nhớ

Nhược điểm: tham số có nguy cơ bị thay đổi giá trị ngoài ý muốn.

Khắc phục nhược điểm này bằng cách dùng từ khoá const (xem VD4).

4. Ví dụ 3: hàm tính chu vi và diện tích hình chữ nhật

Dùng 2 tham số để mang 2 giá trị chu vi và diện tích ra khỏi hàm. 
#include <iostream>
using namespace std;

void cvdthcn(int a, int b, int &cv, int &dt)
{
    cv = 2*(a+b);
    dt = a*b;
}

int main()
{
    int x = 10, y=5, chuvi, dientich;
    cvdthcn(x,y,chuvi,dientich);
    cout<<"Chu vi: "<<chuvi<<endl;
    cout<<"Dien tich: "<<dientich<<endl;
    return 0;
}

Tham số mang giá trị vào hàm: a, b (tham số giá trị)

Tham số mang giá trị ra khỏi hàm: cv, dt (tham số biến)

5. Ví dụ 4: hàm trả về xâu ngược của xâu s

#include <iostream>
#include <cstring>
using namespace std;

string xaunguoc(string const& s)
{
    string p;
    for (int i=0; i<s.length(); i++) p=s[i]+p;
    return p;
}

int main()
{
    string x = "abcd";
    cout<<xaunguoc(x)<<endl;
    return 0;
}

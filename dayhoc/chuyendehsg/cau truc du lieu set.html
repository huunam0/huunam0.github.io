Cấu trúc dữ liệu Set

Khái niệm:

SET là kiểu dữ liệu để lưu các phần tử cùng kiểu và mỗi phần tử là duy nhất.

Đặc trưng của set:

  • Mỗi phần tử là duy nhất
  • Các phần tử được sắp xếp tăng dần
  • Các phần tử không sửa đổi được (chỉ thêm và xoá)

Khai báo:

set <kiểu dữ liệu> biến;

Ví dụ: set <int> t;

hoặc set <string> h;

…

Các hàm:

  • set.begin() là con trỏ trỏ đến phần tử đầu tiên của set
  • set.end() là con trỏ trỏ đến sau phần tử cuối cùng của set (không thuộc set)
  • set.size() trả về số lượng phần tử đang có trong set
  • set.empty() kiểm tra set có rỗng không (tương đương set.size()==0)
  • set.insert() thêm 1 phần tử vào set
  • set.remove() xoá một hoặc nhiều phần tử ra khỏi set
  • set.clear() xoá hết các phần tử của set
  • set.count() đếm số lần xuất hiện của 1 giá trị trong set. Kết quả là 0 hoặc 1.
  • set.find() tìm một phần tử trong set. Trả về con trỏ trỏ đến vị trí của phần tử cần tìm.

Duyệt set

Cho set <int> t; In ra các phần tử của set t:

set<int> t;
set<int>::iterator it;
for (it=t.begin(); it!=t.end(); it++)
{
    cout << *it;
}

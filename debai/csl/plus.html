#1. PLUS - A cộng B

Cho hai số nguyên và . Tính và in ra kết quả biểu thức .

Dữ liệu vào:

  • Một dòng duy nhất chứa hai số nguyên được ghi cách nhau một dấu cách.

Dữ liệu ra:

  • Một số nguyên duy nhất là kết quả của phép tính .

Ví dụ:

Dữ liệu vào:
2 3
Dữ liệu ra:
5

Giới hạn:

  • số test có

Code mẫu:

Bạn phải nhập/xuất đúng theo đề bài yêu cầu. Không "thông báo" khi nhập/xuất.

C++, C++11, C++14, C++17
#include <iostream>
using namespace std;
int main() {
    int A, B;
    cin >> A >> B;
    cout << A + B;
    return 0;
}
C
#include <stdio.h>
int main() {
    int A, B;
    scanf("%d %d\n", &A, &B);
    printf("%d\n", B + A);
    return 0;
}

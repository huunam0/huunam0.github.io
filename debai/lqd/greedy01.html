Căn bậc 2 của mũ 2

Cho dãy số dãy số nguyên \(n\) phần tử \(A_1, A_2, ..., A_n\). Hãy tính giá trị \(X\) sau đây.

Dữ liệu vào:

  • Số nguyên dương \(n (n \le 10^5)\) - số phần tử.
  • \(n\) số nguyên \(A_1, A_2, ..., A_n (A_i^2 \le 10^{18})\)

Dữ liệu ra:

  • Giá trị của \(X\).

Input

3
3 -7 0

Output

3

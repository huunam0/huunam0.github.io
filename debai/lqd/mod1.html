Module 1

Bạn được cho 3 số nguyên dương \(a\), \(b\), \(c\). Hãy tính 3 phần dư của các giá trị \((a + b)\) , \((a - b)\), \((a \times b)\). khi chia cho c

Input

  • Dòng đầu tiền : 3 số nguyên dương \(a, b, c\) \((a,b \leq 10^{16}, c \leq 10^9)\)

Output

  • Chứa 3 giá trị trên.

Sample Input 1

1 2 3

Sample Output 1

0 2 2
